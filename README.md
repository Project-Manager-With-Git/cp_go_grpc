# cp_go_grpc

go语言grpc项目的组件

## 组件包括

| 组件名           | 说明                       |
| ---------------- | -------------------------- |
| `sdk`     | grpc的客户端sdk模板"       |
| `serv`           | grpc的服务端模板           |
| `main`           | 入口文件                   |
| `service_schema` | 项目定义service的proto文件 |
